# chatapp

チャットアプリのソースコード

## アプリの起動

1. 次のコマンドを実行します

   ```bash
   # アプリのディレクトリに移動
   cd ~/hackathon/chatapp
   # アプリの開発で使用するライブラリをインストール（初回のみ）
   npm ci
   # 起動コマンドを実行
   npm run start
   ```

2. `http://サーバのIPアドレス:ポート番号/` にブラウザでアクセスします

   例：`http://127.0.0.1:3000/`

## 機能

### 基本要件

#### ログイン画面

- ユーザ名が入力し「入室する」を押す

#### チャット画面

- 「投稿」ボタンでメッセージを投稿する
  - 投稿されたメッセージは自分を含め、すべてのクライアントに投稿者名とともに表示される
  - 例） ○○○○ さん：（投稿文）
  - 自分の投稿は右側に、相手の投稿は左側に表示されます。
  
- 「メモ」ボタンでメモを投稿する
  - 投稿されたメモは自分にだけ表示される
  - 例） ○○○○ さんのメモ：（投稿文）
- 投稿は古い投稿から順に表示される
- ユーザの入退室時に自分を除いた他のクライアントに入退室のメッセージが表示される
  - 入室の例） ○○○○ さんが入室しました
  - 退室の例） ○○○○ さんが退室しました
